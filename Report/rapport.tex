\chapter{Introduction Générale}
Le projet \textbf{SmartCane} (SIRIUS) est une solution technologique innovante dédiée à l'amélioration de l'autonomie et de la sécurité des personnes malvoyantes. Grâce à l'intégration de technologies IoT, Cloud et Mobiles, SmartCane offre un système de surveillance et d'assistance en temps réel.

\chapter{Analyse et Conception}
\section{Objectifs du projet}
\begin{itemize}
    \item Localisation GPS en temps réel.
    \item Détection de chutes et situations d'urgence.
    \item Interface de gestion pour les centres de supervision (Dashboard).
    \item Site web vitrine pour la présentation des services.
\end{itemize}

\section{Architecture Globale}
L'architecture repose sur trois piliers :
\begin{enumerate}
    \item \textbf{Hardware (IoT) :} Basé sur l'ESP32, il collecte les données des capteurs et les envoie via Firebase.
    \item \textbf{Backend :} API développée avec FastAPI et MySQL pour la gestion persistante des utilisateurs et des historiques.
    \item \textbf{Frontend :} 
    \begin{itemize}
        \item Dashboard Flutter pour le monitoring en temps réel.
        \item Site Web HTML/Tailwind pour la présentation commerciale.
    \end{itemize}
\end{enumerate}

\chapter{Réalisation Technique}

\section{Backend : FastAPI \& MySQL}
Le backend assure la gestion des comptes, des locations et de l'historique des alertes.

\section{Site Web Vitrine}
Utilisation de Tailwind CSS pour un design moderne et responsive. Il permet une présentation fluide des trois modèles : \textit{Smart Lite}, \textit{Smart Pro} et \textit{Smart Elite}.

\chapter{Architecture Logicielle}
\section{Modélisation de la Base de Données}
Le système utilise une base de données MySQL structurée autour des entités suivantes :
\begin{itemize}
    \item \textbf{Utilisateurs :} Gestion des clients (malvoyants) et du personnel (staff/admin).
    \item \textbf{Cannes :} Inventaire des dispositifs disponibles et leur état.
    \item \textbf{Alertes :} Journalisation des incidents SOS et HELP avec coordonnées GPS.
    \item \textbf{Locations \& Abonnements :} Suivi commercial des équipements.
\end{itemize}

\section{Services Backend}
L'API FastAPI est découpée en services spécialisés :
\begin{itemize}
    \item \texttt{AlertService} : Gère la logique de résolution et de synchronisation.
    \item \texttt{AuthService} : Sécurise l'accès via hachage Bcrypt et gestion de mots de passe.
    \item \texttt{EmailService} : Automatise les réponses aux messages clients via SMTP.
\end{itemize}

\chapter{Conclusion et Perspectives}
Le projet SmartCane démontre l'efficacité de l'Internet des Objets au service du social. Les perspectives d'évolution incluent l'intégration de l'Intelligence Artificielle pour la reconnaissance d'obstacles via caméra.
